#ReadUP

A Django + HTMX web application for tracking and managing books.

## Live Demo
https://readup.onrender.com  

## Features
- Add and manage books
- Dynamic UI updates with HTMX
- Clean, modular Django structure

## Tech Stack
- Django
- HTMX
- Bootstrap


## Run Locally
```bash
git clone https://github.com/Malcom-Yingwani/ReadUP.git
cd ReadUP
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
